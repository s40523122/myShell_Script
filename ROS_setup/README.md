# 安裝 ROS
1. 取得執行指令集</br>
  `wget https://raw.githubusercontent.com/s40523122/myShell_Script/main/ROS_setup/ros_install.sh`
2. 給予執行權限 </br> `chmod +x ros_install.sh`
3.  輸入指令自動安裝 </br> `./ros_install.sh`
